# 快速开始 - Web UI

## 启动 UI

```bash
python run_ui.py
```

浏览器会自动打开 http://localhost:8501

## 界面说明

### 主界面

1. **需求输入区域**（左侧）
   - 选择输入方式：文本输入 / 文件上传 / URL
   - 输入或上传您的需求描述
   - 点击"生成代码"按钮开始

2. **状态显示区域**（右侧）
   - 显示当前工作流状态
   - 显示最终评分和生成文件数量

3. **日志区域**（底部）
   - 实时显示工作流执行日志
   - 查看每个阶段的详细信息

### 侧边栏配置

- **Anthropic API Key**: 输入您的 API 密钥
- **Project Directory**: 设置代码生成目录
- **Review Mode**: 选择自动或手动审查
- **Pause for Review**: 是否在设计阶段暂停

## 示例需求

```
开发一个简单的待办事项API:

功能需求:
1. 创建待办事项
2. 查看待办事项列表
3. 更新待办事项状态
4. 删除待办事项

技术要求:
- 后端: Python + FastAPI
- 数据库: SQLite
- 返回JSON格式

非功能需求:
- 代码简洁易读
- 包含单元测试
- 包含API文档
```

## 注意事项

- 首次使用需要在侧边栏输入 API Key
- 工作流执行需要几分钟时间，请耐心等待
- 生成的代码会保存在指定的项目目录中
- 可以在 `项目目录/docs/` 中查看详细报告

## 故障排除

如果遇到问题，请检查：
1. API Key 是否正确设置
2. 网络连接是否正常
3. 项目目录是否有写入权限
4. 查看日志输出了解具体错误
