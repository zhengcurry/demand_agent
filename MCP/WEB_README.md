# 飞书项目管理 - Web界面使用说明

## 功能特性

- ✅ 工作项列表查看
- ✅ 三个筛选条件（关联项目、规划版本、所属业务线）
- ✅ 分页浏览
- ✅ 实时数据加载
- ✅ 响应式设计

## 启动方法

### 方式1: 直接运行
```bash
cd MCP
python web_app.py
```

### 方式2: 使用Flask命令
```bash
cd MCP
set FLASK_APP=web_app.py
flask run
```

## 访问地址

启动后访问: http://127.0.0.1:5000

## 使用说明

1. **查看工作项列表**
   - 页面加载后自动显示第1页的工作项

2. **筛选工作项**
   - 选择"关联项目"下拉框筛选特定项目
   - 选择"规划版本"下拉框筛选特定版本
   - 选择"所属业务线"下拉框筛选特定业务线
   - 点击"应用筛选"按钮查看结果

3. **重置筛选**
   - 点击"重置筛选"按钮清除所有筛选条件

4. **分页浏览**
   - 使用"上一页"和"下一页"按钮浏览不同页面
   - 页面顶部显示当前页码和工作项总数

## 界面说明

### 筛选区域
- **关联项目 (field_c13a6c)**: 筛选特定项目的工作项
- **规划版本 (field_ab3f51)**: 筛选特定版本的工作项
- **所属业务线 (business)**: 筛选特定业务线的工作项

### 工作项表格
显示以下字段：
- 工作项ID
- 名称
- 所属业务线
- 关联项目
- 规划版本
- 优先级（带颜色标识）
- 状态
- 创建者

### 优先级颜色
- 🔴 红色: 重要且紧急
- 🟡 黄色: 重要但不紧急
- 🔵 蓝色: 紧急但不重要
- ⚪ 灰色: 不紧急不重要

## 技术栈

- **后端**: Flask (Python)
- **前端**: HTML + CSS + JavaScript
- **API**: 飞书MCP Server

## 故障排除

### 问题1: 无法启动
- 确保已安装Flask: `pip install flask`
- 检查端口5000是否被占用

### 问题2: 数据加载失败
- 检查MCP Server配置是否正确
- 确认网络连接正常
- 查看控制台错误信息

### 问题3: 筛选不生效
- 点击"应用筛选"按钮
- 检查是否选择了有效的筛选条件
